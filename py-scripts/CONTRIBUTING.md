# K8s Toolbelt(scripts)

## Setup
Use Python 3.9 (recommend 3.9.10)

Dependency import
```
$ pip3 install -r requirements.txt
```

# Test
Use pytest
```
$ pytest tests/

or

$ pytest --pdb test/
```

# Rule [recommend]
Lint: [flake8](https://flake8.pycqa.org/en/latest/)<br/>
Optional Formatter: [black](https://black.readthedocs.io/en/stable/)
